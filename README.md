# Netflix-Clone
This app is a clone of Netflix app.

- 100% Programmatic UI
- 3rd party libraries
  - SDWebImage (Swift Package Manager)
- Core Data
- APIs used in the project
  - TMDB API
  - YouTube Data API v3
